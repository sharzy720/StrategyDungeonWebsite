RPCS3 install guide

Prerequisites

Installing

Go to (RPCS3.net hyperlink) to download the RPCS3 zip. Once finished downloading unzip the folder and move it to where you want to install RPCS3. After moving the folder open it and double click ‘rpcs3.exe’ to start the emulator.

Once RPCS3 is open it is time to add the PS3 bios, you can get the PS3 bios from playstation.com. If the download button is not working right click it and click ‘save link as’. Save the bios somewhere you will be able to find it later.

To install the bios go to (file -> install firmware) When the pop-up displays add your downloaded bios files either by dragging them onto the window or by using to “install firmware” button and selecting the file.

After adding the PS3 bios, RPCS3 is fully set up and able to run games.
